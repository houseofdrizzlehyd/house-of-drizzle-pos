
## Usability update

- Async form actions and navigation now display loading feedback and prevent duplicate clicks.
- The desktop sidebar can collapse to an icon rail and remembers the preference.
- Smaller screens use a slide-out mobile navigation drawer.
- Printed receipts use a clean Arial/Rubik-style font instead of a monospace thermal font.
- Admins and Super Admins can bulk-create categories and create/update products through Excel Import.
- The Excel template is available at `public/product-import-template.xlsx`.
